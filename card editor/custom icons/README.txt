Custom power/status icons

Put custom icon image files in this folder.

Supported file types:
- .png
- .jpg
- .jpeg
- .webp

How to use them in Card Editor:
1. Turn the effect into a power.
2. Set Status Icon Mode to Custom.
3. In Status Icon, type the small icon file name, for example:
   my_power_small.png
4. In Status Big Icon, type the large icon file name if you have one.

Notes:
- You can type just the file name. You do not need to use res://.
- If Status Big Icon is empty, the editor will fall back to the small icon.
- Old res:// paths still work if you already use them.
- This folder sits next to the card art folder on purpose.
