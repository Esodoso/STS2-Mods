# Defect Full-Art Card Pack

Proof-of-concept full-art visual pack for Defect cards in Slay the Spire 2.

This pack imports through Card Art Editor and covers 92 Defect card portraits. It is visual only: it does not add cards, change card effects, rebalance cards, or alter gameplay.

## Requirements

- Slay the Spire 2
- Card Art Editor by p1585:

```text
https://www.nexusmods.com/slaythespire2/mods/293
```

## Installation

1. Install Card Art Editor.
2. Extract this archive.
3. Start Slay the Spire 2 with mods enabled.
4. Open the Compendium.
5. Right-click any card to open Card Art Editor.
6. Use Card Art Editor's import option.
7. Select:

```text
defect-full-art.cardartpack.json
```

8. Apply/import the pack.
9. Check Defect cards in the Compendium or in a run.

## Included File

```text
defect-full-art.cardartpack.json
```

This is the Card Art Editor import bundle. Import it through Card Art Editor; do not place it directly into the game files as a texture replacement mod.

## Notes

- This is a proof-of-concept visual pack.
- Some generated continuations may still need polish.
- Future Slay the Spire 2 updates may change card IDs, asset paths, or rendering behavior.
