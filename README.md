Dopo aver clonato i file avviare install.ps1
aprendo terminale e scrivendo: .\install.ps1
quando richiesto il percorso copiarlo dalla cartella "mods" di STS2, esempio: "C:\Steam\steamapps\common\Slay the Spire 2\mods" ( se non esiste cartella "mods" basta crearla)
Successivamente ad ogni aggiornamento basta aprire terminale da dentro questa cartella ed eseguire: git pull e avviare .\install.ps1
